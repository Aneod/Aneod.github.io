Foire Aux Questions (FAQ)

Un site web coûte-t-il vraiment le prix indiqué ?

Le prix indiqué pour un site web est son prix minimum. Néanmoins celui-ci peut augmenter. Dans le cas où les exigences d'un clients auraient un certain niveau de complexité dans leurs mises en oeuvre, nous devrons apporter davantages de notre temps et de nos efforts pour satisfaire la demande du client jusqu'au bout. Mais afin de rester rentable dans notre démarche, le prix augmentera alors d'une valeur déterminée au cas par cas, généralement aux heures supplémentaire nécessaires. Si cette question reste pour vous une inquiétude, n'hesitez pas à nous contacter pour en discuter.

J'ai déjà un site web, est-il possible de le reprendre au lieu d'en crée un autre ?

C'est possible dans certain cas, en particulier si le site n'a pas été conçu sur la base d'une grandes plateforme de création de site web. Dans tout les cas cette question nécessite une analyse de votre site web en question.

Suis-je propriétaire de mon site web ?

Oui, et à 100%. Le site vous appartient dans sa globalité. Nom de domaine inclus.
